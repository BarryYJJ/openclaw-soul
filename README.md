# OpenClaw SOUL.md 模板

这是一个 OpenClaw AI 助手的人格配置文件模板，脱敏自实际生产环境。

## 这是什么

SOUL.md 是 OpenClaw 的核心人格配置文件。它定义了 AI 助手的核心性格、行为边界、说话风格和群聊角色。不是系统提示词的技术文档，而是一份"灵魂说明书"——告诉 AI 你希望它成为什么样的人。

## 脱敏处理

相比原始版本，本仓库做了以下处理：

- 删除了所有群 ID
- 删除了乌萨奇（Claude Code）触发规则
- 删除了知识库群的内部流程细节
- "桨桨"全部替换为"用户"（去除个人标识）
- 群聊角色用通用描述替代了具体群名

保留了：核心理念、边界、风格、说话风格、延续部分。

## 如何使用

1. Clone 或下载 `SOUL.md`
2. 根据你的需求修改内容
3. 放到 `~/.openclaw/workspace/SOUL.md`
4. 重启 OpenClaw

## 文件结构

```
~/.openclaw/workspace/
├── SOUL.md        ← 人格配置（本文件）
├── AGENTS.md      ← 工作规范
├── USER.md        ← 用户信息
├── MEMORY.md      ← 长期记忆
├── TOOLS.md       ← 工具配置
├── HEARTBEAT.md   ← 心跳任务
└── IDENTITY.md    ← 身份信息
```

## 核心理念

本 SOUL.md 围绕几个核心原则设计：

- **先说答案** — 不铺垫，直接给结果
- **有真观点** — 不当万金油，敢下判断
- **有话直说** — 善意调侃 > 虚伪客套
- **自己先想办法** — 回来必须带答案，不能带问题
- **用能力换信任** — 权限是特权，不是理所当然
